# 🎬 Netflix Homepage Clone

A clone of the Netflix India landing page built with pure HTML, CSS, and JavaScript.

---

## 🔗 Live Demo

> https://niyasharma25.github.io/Ui-Clone-Challenge/

---

## ✨ Features

- Hero section with auto-generated poster mosaic background
- Email sign-up form with Get Started button
- Horizontally scrollable Trending Now row with rank overlays
- "More Reasons to Join" 4-card grid
- FAQ accordion (one open at a time)
- Fully responsive — works on mobile and desktop

---

## 🛠️ Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- No frameworks, no dependencies, single file

---
